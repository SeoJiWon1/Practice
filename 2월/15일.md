# CSS
* CSS(Cascading Style Sheet)
* HTML 문서의 색이나 모양 등 외관을 꾸미는 언어
* CSS로 작성된 코드를 스타일 시트(style sheet)라고 부름
## CSS의 기능
* 색상과 배경
* 텍스트
* 폰트
* 박스 모델(Box Model)
* 비주얼 포맷 및 효과
* 리스트
* 테이블
* 사용자 인터페이스

### class 셀렉터
* 점(.)으로 시작하는 이름의 셀렉터
